Help
   Butlr is a command-line for the web.
   Like any command-line, butlr has specific syntax.
   For a list of available commands, type 'help' from butlr.
   Most commands have parameters, additional information you need to supply.
   For an explanation of a command's specific syntax type 'help [command name]'.
   For any further questions/feedback, email contact@butlr.me.
